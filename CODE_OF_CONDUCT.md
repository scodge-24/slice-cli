# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, personal attacks, and
discriminatory behavior are not tolerated. We want this to be a welcoming space
for contributors of all backgrounds and experience levels.

Report unacceptable behavior to the maintainer at scott.hodge@live.co.uk. Reports
will be reviewed and handled confidentially.
